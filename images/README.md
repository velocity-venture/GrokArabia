# Images Folder

## hero-bg.jpg

**Important**: You need to add the hero background image here.

The hero-bg.jpg file should be:
- A cyberpunk-style cityscape with neon green accents
- Resolution: At least 1920x1080px
- File size: Optimized for web (under 500KB recommended)

### How to get the image:

1. **From the hub**: Download the hero-bg.jpg file from the GrokArabia hub
2. **Or use the URL**: https://www.genspark.ai/api/files/s/1wIj3tm2
3. **Or create your own**: Use a similar cyberpunk/tech themed background

### After adding the image:

```bash
git add images/hero-bg.jpg
git commit -m "Add hero background image"
git push
```
